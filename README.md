# 🎓 Student Management System

A desktop application for managing student records, built with **C#** and **Windows Forms**.

## 📷 Screenshot

![Student Management System](image/IDcard.png)

## 📖 About the Project

This project is a simple desktop application designed to manage student information through an easy-to-use interface.

Users can add new students, view their information, delete records, and display a student ID card by selecting a student from the list.

The project helped me improve my understanding of Windows Forms development, event-driven programming, and desktop application design.

## ✨ Features

- ➕ Add new student records.
- 🗑️ Delete selected students with confirmation.
- 📋 Display students using ListView.
- 🆔 View a student ID card by double-clicking a student.
- 🖨️ Open a separate form to preview and print the student ID card.
- 👤 Display different avatars for male and female students.
- 📱 Input validation using MaskedTextBox for ID and phone number.
- 🔄 Refresh the input fields and ID card information.
